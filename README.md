# 数字当てゲーム

「数字当てゲーム」を作ってJavaScriptの基礎を学ぼう！

## Demo

https://kazunoko1606.github.io/guess_number/

## Description

1~20までの数字を当てるシンプルなゲームです。  
ヒントを頼りに正解の数字を予想していきます。  
より少ない回数で正解すると高得点になります。  

## Note

starterフォルダから必要なファイルをダウンロードできます。
